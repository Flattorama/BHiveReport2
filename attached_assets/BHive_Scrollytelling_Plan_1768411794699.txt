# BHive 2025 Annual Report: Scrollytelling Development Plan
## "Resilience & Refinement" — An Interactive Web Experience

---

## I. PROJECT OVERVIEW

### Vision Statement
Transform BHive's 2025 Annual Report into an immersive, scroll-driven narrative that visualizes the organization's strategic pivot from volume to quality in response to IRCC regulatory changes. The experience should feel like "entering a hive"—users journey through interconnected hexagonal cells, each revealing a story of adaptation, global impact, and entrepreneurial success.

### Core Narrative Arc
1. **Opening Hook** → Massive impact numbers draw users in
2. **The Challenge** → IRCC cap creates constraints
3. **The Pivot** → Quality over quantity transformation
4. **Global Reach** → International connections and delegations
5. **The Ecosystem** → Cohorts, programming, community
6. **Growth Story** → Digital metrics explosion
7. **Timeline Journey** → Year of milestones
8. **Future Vision** → What's next for BHive

---

## II. TECHNICAL STACK

### Core Framework
```
React 18+ (with Server Components for SEO)
Next.js 14 (App Router)
TypeScript
Tailwind CSS 3.4+
```

### Animation & Scroll Libraries
```
GSAP (GreenSock) — Core scroll animations
ScrollTrigger — Scroll-based triggering
Framer Motion — Component transitions
Lenis — Smooth scroll behavior
```

### Data Visualization
```
D3.js — Custom visualizations (globe, funnel)
Recharts — Charts and graphs
Three.js + React Three Fiber — 3D globe
react-spring — Physics-based animations
```

### Additional Dependencies
```
react-intersection-observer — Scroll detection
canvas-confetti — Celebration effects
react-countup — Animated number counters
mapbox-gl OR deck.gl — Alternative map solution
```

## V. INTERACTION & ANIMATION SPECIFICATIONS

### Scroll-Triggered Animations

| Element | Trigger Point | Animation |
|---------|---------------|-----------|
| Section headers | 20% viewport | Fade up + slide |
| Counter numbers | 30% viewport | Count up |
| Charts/graphs | 40% viewport | Draw path |
| Cards | Staggered | Fade + scale |
| Globe | Section enter | Start rotation |
| Timeline | Section enter | Reveal markers |

### Micro-interactions
- **Buttons**: Scale 1.05 on hover, bhive-orange glow
- **Cards**: Subtle lift (translateY -4px) + shadow
- **Hexagons**: Pulse animation on hover
- **Links**: Underline draw animation
- **Numbers**: Slight bounce on count complete

### Performance Targets
- **First Contentful Paint**: < 1.5s
- **Largest Contentful Paint**: < 2.5s
- **Time to Interactive**: < 3.5s
- **Cumulative Layout Shift**: < 0.1
